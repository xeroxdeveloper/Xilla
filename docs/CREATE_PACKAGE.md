# Создание пакетов (Packages) для Xilla Userbot

*Доступно с версии Alpaca B1*

Пакеты — это более сложный уровень расширения Xilla. В отличие от модулей (которые обычно состоят из одного файла), пакеты могут включать в себя:
- Множество взаимосвязанных модулей
- Свои собственные библиотеки (libs)
- Статические файлы (картинки, данные, конфигурации)
- Сложные обработчики (watchers)
- Изменение поведения ядра (hooking)

## Как работает система пакетов?

На данный момент система находится в стадии Beta. Встроенный пакет **Main** отвечает за загрузку и парсинг сторонних пакетов.

Для создания пакета вам понадобится создать репозиторий с определенной структурой:

```
MyAwesomePackage/
├── manifest.json
├── package_init.py
├── modules/
│   ├── my_advanced_mod.py
│   └── helper_mod.py
├── libs/
│   └── my_custom_api.py
└── assets/
    └── icon.png
```

Подробная спецификация `manifest.json` и API пакетов будет опубликована в релизе **Alpaca 1.0**.

Пока что вы можете изучать исходный код ядра (`xilla/loader.py`) для понимания того, как Xilla импортирует файлы!
