## 🌟 Особенности

- **🚀 Молниеносная скорость и оптимизация**: В версии `Alpaca B3` юзербот потребляет **в 500 раз меньше оперативной памяти** благодаря встроенному `memory_optimizer` и агрессивному Garbage Collector-у. Идеально для слабых VDS или Android (Termux).
- **☀️ Невероятная эстетика**: Фирменный сине-голубой градиентный стиль терминала и локально генерируемые баннеры (`Pillow`) для всех системных команд. Бот буквально рисует красивые картинки для каждого ответа!
- **🧠 Интегрированный ИИ (XillaAI)**: Встроенный доступ к `Gemini-3.1-Pro` для ответов на вопросы и бесплатная генерация изображений (1024x1024) прямо из чата. Бесплатно и без ключей.
- **📦 Система Пакетов (Alpaca)**: Полностью переработанный Конфиг Хаб (команда `.cfg`) со вкладками для настройки Модулей, Библиотек и Пакетов.
- **🛡️ 100% стабильность**: Все известные краши устранены. Добавлены блокировки от зависания базы данных (SQLite lock), умный перехват `CTRL+C` и обход `EOFError`.
- **🗜️ XillaZip & XillaPack**: Встроенные утилиты для работы с файлами. Архивация любых файлов и упаковка питон-модулей в любой текстовый формат.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы (Linux, MacOS, Termux).

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   python3 xilla/xilla.py
   ```
   *При первом запуске вам потребуется ввести номер телефона и код из Telegram для авторизации.*

### Запуск одной командой
Для большего удобства Xilla автоматически настраивает себя. После первой установки вы можете просто запускать юзербота глобальной командой в терминале:
```bash
xilla
```

## 🛠 Возможности и модули

Xilla поставляется с набором мощных системных модулей, которые обеспечивают базовый, но невероятно красивый функционал:

| Команда | Описание |
| :--- | :--- |
| `.help` | Выводит интерактивный список всех модулей с **Pillow-баннером**. |
| `.info` | Показывает подробную статус-карточку юзербота (Версия, Аптайм, RAM/CPU) с сгенерированной обложкой. |
| `.ping` | Проверка задержки и статуса серверов (включает миллисекунды прямо на картинке). |
| `.cfg` | Открывает **Xilla Configuration Hub** — интерактивное меню настроек прямо в Telegram. |
| `.ai <вопрос>` | Задать вопрос встроенному ИИ `Gemini-3.1-Pro`. |
| `.igen <описание>` | Сгенерировать квадратную картинку по вашему описанию. |
| `.zip <реплай>` | Запаковать файл(ы) или директорию в `.zip`, `.tar`, `.tgz`, `.bz2` и другие форматы. |
| `.pack <ext>` | Инновационная фича. Упаковывает Python-модуль (`.py`) с использованием `zlib+base64` в любой текстовый формат (например `.txt`, `.lua`, `.rb`) для удобной передачи. |
| `.unpack` | Распаковывает Xilla Pack обратно в рабочий модуль. |
| `.update` | Безопасное обновление юзербота прямо из Telegram (перезагрузка ядра). |

*Заметка: Xilla использует механизм принудительного обновления. Если на GitHub выходит критический патч, бот попросит вас прописать `.update` перед продолжением работы.*
