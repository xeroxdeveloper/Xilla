## 🌟 Особенности

- **🚀 Молниеносная скорость и оптимизация**: В версии `Alpaca B3` юзербот потребляет **в 500 раз меньше оперативной памяти** благодаря встроенному `memory_optimizer` и агрессивному Garbage Collector-у. Идеально для слабых VDS или Termux.
- **☀️ Невероятная эстетика**: Фирменный сине-градиентный стиль терминала и красивые плашки в Telegram.
- **🧠 Интегрированный ИИ (XillaAI)**: Встроенный доступ к `Gemini-3.1-Pro` для ответов на вопросы и генерация картинок прямо из чата. Бесплатно и без ключей.
- **📦 Новая система Пакетов (Alpaca)**: Полностью переработанный Конфиг Хаб (команда `.cfg`) со вкладками для Модулей, Библиотек и Пакетов.
- **🛡️ 100% стабильность**: Все известные краши устранены. Добавлены блокировки от зависания базы данных (SQLite lock), умный перехват `CTRL+C` и обход `EOFError`.
- **🗜️ XillaZip**: Встроенный супер-архиватор. Создавайте `.zip`, `.tar`, `.gz` прямо ответом на файлы в Telegram.

## ⚙️ Установка

**Для Linux / Termux / MacOS:**
```bash
git clone https://github.com/xeroxdeveloper/Xilla.git
cd Xilla
pip install -r requirements.txt
python3 xilla/xilla.py
```

### Запуск одной командой
Xilla автоматически создает удобный бинарник для запуска. После первой установки вы можете просто запускать юзербота командой `xilla`, без лишних аргументов!

## 📚 Основные команды

| Команда | Описание |
| :--- | :--- |
| `.help` | Выводит красивый список всех установленных модулей и команд. |
| `.info` | Показывает подробную информацию о юзерботе (Версия, Ветка, Аптайм, RAM/CPU) с красивым баннером. |
| `.ping` | Проверка задержки и статуса серверов. |
| `.cfg` | Открывает **Xilla Configuration Hub** — интерактивное меню настроек прямо в Telegram. |
| `.ai <вопрос>` | Задать вопрос встроенному ИИ `Gemini-3.1-Pro`. |
| `.igen <описание>` | Сгенерировать квадратную картинку (1024x1024) по описанию. |
| `.zip <реплай>` | Запаковать файл(ы) в `.zip` архив. |
| `.update` | Автоматически обновить ядро юзербота до последней версии с GitHub. |

*Xilla поддерживает десятки других полезных команд для администрирования чатов, автоматизации рутины и веселья!*
