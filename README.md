## О проекте

Xilla — это современный юзербот для Telegram, созданный с упором на скорость, стабильность и эстетику. В отличие от других юзерботов, Xilla полностью отказалась от устаревшего веб-интерфейса в пользу молниеносной консоли и встроенного AI.

## Установка

Вы можете установить Xilla на Linux, MacOS или в Termux (Android).

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   python3 xilla/xilla.py
   ```
   *При первом запуске вам потребуется ввести номер телефона и код из Telegram для авторизации.*

## Основной функционал

* **XillaAI (`.ai`, `.igen`)** — Встроенный искусственный интеллект (Gemini 3.1) для ответов на вопросы и бесплатная генерация качественных изображений (1024x1024) прямо из чата. Не требует API ключей!
* **XillaZip (`.zip`, `.tar`, `.tgz`)** — Продвинутый архиватор файлов. Просто ответьте на файл или папку, и бот запакует её в нужный формат.
* **XillaConfig (`.cfg`)** — Интерактивный хаб настроек с красивыми инлайн-кнопками. Позволяет настраивать модули и новые "Пакеты" не выходя из Telegram.
* **Мандаторные обновления (`.update`)** — Встроенная система защиты. Бот автоматически проверяет GitHub на наличие критических обновлений и блокирует команды до их установки, гарантируя, что вы всегда используете самую безопасную версию.
