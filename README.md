## 🌟 Философия Xilla A1

В версии **A1** юзербот был полностью переписан с абсолютного нуля. В нём нет ни единой строчки старого или чужого кода.

- **Сверхлёгкое ядро**: Использует `herokutl` (современный форк Telethon) и нативную архитектуру загрузчика. Бот весит в 10 раз меньше, чем его аналоги.
- **Никакого мусора**: Устаревшие библиотеки, бесполезные веб-панели и громоздкие базы данных вырезаны. Все настройки хранятся в элегантном `xilla.json`.
- **Plug-and-Play совместимость**: Поддержка Legacy-модулей реализована через `sys.modules` хук. Скрипты, написанные под Hikka или Heroku, всё ещё могут загружаться в папку `plugins`.
- **Максимальная конфиденциальность**: Мы используем официальное API Gemini для ИИ. Ваш ключ безопасно хранится локально в вашем файле конфигурации и никогда не передается на сторонние прослойки.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы.

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   ./xilla.py
   ```
   *При первом запуске вам потребуется ввести API Hash, API ID и номер телефона.*

## 🛠 Создание модулей

Создавать модули для Xilla A1 невероятно просто:

```python
from xilla import Module, command

class MyMod(Module):
    """Мой первый модуль"""
    
    @command("hi")
    async def say_hi(self, message):
        """Сказать привет"""
        await message.edit("<b>☀️ Привет!</b>", parse_mode="HTML")
```
Просто поместите файл в папку `plugins/`, и юзербот автоматически подхватит его при старте.
