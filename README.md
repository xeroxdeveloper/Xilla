## 🌟 Философия Xilla A1

В версии **A1** юзербот был полностью переписан с абсолютного нуля. В нём нет ни единой строчки старого или чужого кода.

- **Сверхлёгкое ядро**: Использует `herokutl`.
- **Никакого мусора**: Устаревшие библиотеки вырезаны.
- **Plug-and-Play совместимость**: Поддержка Legacy-модулей реализована через `sys.modules`.
- **Модуль Installer (.install)**: Загрузка сторонних модулей по ссылке с GitHub прямо из чата.
- **Асинхронная очередь задач**: Команды не блокируют друг друга при медленном интернете.
- **Auto-Backup**: Автоматическое резервное копирование настроек (`xilla.json`) в Избранное каждые 24 часа.
- **Crash Protection**: Виртуальная изоляция модулей. Ошибка синтаксиса больше не "убивает" юзербота.
- **Песочница Eval (.eval)**: Выполнение любого Python-кода с перехватом консольного вывода.
- **I18n Localizer**: Встроенная мультиязычность.
- **Media Cache Engine**: Юзербот интеллектуально кэширует скачанные картинки.
- **Модульный SQLite3**: Встроенная микро-база данных только по запросу.
- **Pillow Config (.config)**: Красивый графический терминал управления настройками.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы.

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   ./xilla.py
   ```
   *При первом запуске вам потребуется ввести API Hash, API ID и номер телефона.*

## 🛠 Создание модулей

Создавать модули для Xilla A1 невероятно просто:

```python
from xilla import Module, command

class MyMod(Module):
    """Мой первый модуль"""
    
    @command("hi")
    async def say_hi(self, message):
        """Сказать привет"""
        await message.edit("<b>☀️ Привет!</b>", parse_mode="HTML")
```
Просто поместите файл в папку `plugins/`, и юзербот автоматически подхватит его при старте.
