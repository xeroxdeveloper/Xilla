## 🌟 Особенности Xilla

Xilla **написана с нуля** без остатков старого кода и мусора прошлых поколений, чтобы стать самым быстрым юзерботом в Telegram:

- **🧩 Абсолютная совместимость (Hikka, GeekTG, Heroku, FTG)**: Умная система ядра `compat` позволяет Xilla "на лету" подменять старые импорты. Вы можете скачать и запустить модули от любых других юзерботов без переписывания их кода!
- **⚡ Система Плагинов и Пакетов**: В дополнение к сложным Пакетам, добавлена простая папка `plugins/`, куда вы можете просто скидывать чужие скрипты. Они никогда не сотрутся при обновлении юзербота.
- **🎨 Встроенный графический движок (Pillow Banners)**: Больше никаких скучных текстов. Xilla рисует потрясающие градиентные баннеры для системных команд, которые кэшируются в RAM (`_BANNER_CACHE`), ускоряя отрисовку в 10 раз!
- **🚀 Молниеносная оптимизация (GC Engine)**: Благодаря агрессивному Garbage Collector-у и умному планировщику, потребление оперативной памяти снижено **в 500 раз** по сравнению со старыми юзерботами. Бот идеально работает на слабых VDS.
- **🧠 XillaAI (Gemini)**: Мощный модуль искусственного интеллекта. Требует API ключ, который безопасно скрывается за `*` в `.cfg`. Интеллектуальные ответы на вопросы и генерация изображений.
- **📦 XillaPack**: Уникальный архиватор исходного кода. Позволяет зашифровать любой `.py` модуль сжатием `zlib` и спрятать его в простой текстовый формат (например `.txt` или `.rb`) для удобной и безопасной передачи.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы (Linux, MacOS, Android/Termux).

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   python3 xilla/xilla.py
   ```

### Быстрый запуск
После первой установки вы можете просто запускать юзербота глобальной командой в терминале без лишних аргументов:
```bash
xilla
```

## 🛠 Основные системные модули

Юзербот поставляется без лишнего хлама, только самое необходимое и красивое:

| Команда | Описание |
| :--- | :--- |
| `.help` | Выводит интерактивный список всех модулей с **Pillow-баннером**. |
| `.info` | Статус-карточка юзербота (Версия, Аптайм, RAM/CPU) с сгенерированной обложкой. |
| `.ping` | Проверка задержки серверов Telegram (миллисекунды рендерятся прямо на картинке). |
| `.cfg` | Открывает **Xilla Configuration Hub** — интерактивное inline-меню настроек модулей. API ключи скрыты звездочками! |
| `.ai <вопрос>` | Задать вопрос встроенному ИИ (Требуется ввести ключ Gemini в `.cfg`). |
| `.igen <описание>` | Сгенерировать квадратную картинку по вашему промпту (Бесплатно). |
| `.pack <ext>` | Упаковывает Python-модуль (`.py`) в текстовый формат для удобной передачи. |
| `.unpack` | Распаковывает Xilla Pack обратно в рабочий модуль. |
| `.zip <реплай>` | Запаковать файл или директорию в архив (`.zip`, `.tar`, `.tgz`). |
| `.update` | Безопасное обновление юзербота прямо из Telegram. |
