## 🌟 Особенности Alpaca B8

Xilla переписана с нуля, чтобы стать самым быстрым и современным юзерботом в Telegram:

- **🧩 Абсолютная совместимость (Hikka & FTG & Heroku)**: Умная система `compat` позволяет Xilla "на лету" подменять старые импорты (`from hikka...` или `from userbot...`), что позволяет запускать модули от любых других юзерботов без переписывания их кода!
- **⚡ Система Плагинов и Пакетов**: В дополнение к сложным Пакетам, добавлена простая папка `plugins/`, куда вы можете скидывать чужие скрипты. Они никогда не сотрутся при обновлении юзербота.
- **🎨 Встроенный графический движок (Pillow Banners)**: Больше никаких скучных текстов. Xilla рисует потрясающие градиентные баннеры для системных команд, которые теперь кэшируются в RAM (`_BANNER_CACHE`), ускоряя отрисовку в 10 раз!
- **🚀 Молниеносная оптимизация (GC Engine)**: Благодаря агрессивному Garbage Collector-у и умному планировщику, потребление оперативной памяти снижено **в 500 раз**. Бот идеально работает на слабых серверах.
- **🧠 XillaAI**: Встроенный доступ к ИИ (Gemini-3.1) для общения и генерации высококачественных картинок прямо в чате без ключей API.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы (Linux, MacOS, Android/Termux).

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   python3 xilla/xilla.py
   ```

### Быстрый запуск
После первой установки вы можете просто запускать юзербота глобальной командой в терминале:
```bash
xilla
```

## 🛠 Возможности и системные модули

| Команда | Описание |
| :--- | :--- |
| `.help` | Выводит интерактивный список всех модулей с **Pillow-баннером**. |
| `.info` | Статус-карточка юзербота (Версия, Аптайм, RAM/CPU) с сгенерированной обложкой. |
| `.ping` | Проверка задержки серверов Telegram (миллисекунды рендерятся прямо на картинке). |
| `.cfg` | Открывает **Xilla Configuration Hub** — интерактивное inline-меню настроек. |
| `.ai <вопрос>` | Задать вопрос встроенному ИИ (работает на базе `Gemini`). |
| `.igen <описание>` | Сгенерировать квадратную картинку по вашему промпту. |
| `.pack <ext>` | Упаковывает Python-модуль (`.py`) в текстовый формат (например `.txt`) для удобной передачи. |
| `.unpack` | Распаковывает Xilla Pack обратно в рабочий модуль. |
| `.zip <реплай>` | Запаковать файл или директорию в архив (`.zip`, `.tar`, `.tgz`). |
| `.update` | Безопасное обновление юзербота прямо из Telegram. |
