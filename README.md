## 🌟 Особенности Alpaca B7

Xilla переписана с нуля, чтобы стать самым быстрым и современным юзерботом в Telegram:

- **🎨 Встроенный графический движок (Pillow Banners)**: Больше никаких скучных текстов или нерабочих ссылок. Xilla **на лету** рисует потрясающие градиентные баннеры с кастомным текстом для всех системных команд (включая пинг, инфо и справку)!
- **🚀 Молниеносная оптимизация (GC Engine)**: Благодаря агрессивному Garbage Collector-у и умному планировщику, потребление оперативной памяти снижено **в 500 раз**. Бот не "раздувается" со временем и идеально работает на слабых VDS и Termux.
- **🛡️ 100% стабильность ядра**: Полностью устранена проблема "Застрявших баз данных" (SQLite lock), переписан обработчик `CTRL+C` для моментального выключения, а весь инлайн-движок переведён на `Aiogram 3` с исправлением всех ошибок Pydantic.
- **🧠 XillaAI**: Встроенный доступ к ИИ (Gemini-3.1) для общения и генерации высококачественных картинок прямо в чате. Никаких ключей API не требуется!
- **📦 Система XillaPack**: Уникальный архиватор исходного кода. Позволяет зашифровать любой `.py` модуль сжатием `zlib` и спрятать его в простой текстовый формат (например `.txt` или `.rb`) для удобной и безопасной передачи.

## ⚙️ Установка

Xilla легко устанавливается на любые UNIX-подобные системы (Linux, MacOS, Android/Termux).

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/xeroxdeveloper/Xilla.git
   cd Xilla
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите юзербота:**
   ```bash
   python3 xilla/xilla.py
   ```
   *При первом запуске вам потребуется ввести номер телефона и код из Telegram для авторизации.*

### Быстрый запуск
Xilla автоматически настраивает себя как консольную утилиту. После первой установки вы можете просто запускать юзербота командой в терминале:
```bash
xilla
```

## 🛠 Возможности и системные модули

Xilla поставляется с минималистичным, но невероятно красивым и полезным набором встроенных утилит:

| Команда | Описание |
| :--- | :--- |
| `.help` | Выводит интерактивный список всех модулей с **Pillow-баннером**. |
| `.info` | Статус-карточка юзербота (Версия, Аптайм, RAM/CPU) с сгенерированной обложкой. |
| `.ping` | Проверка задержки серверов Telegram (миллисекунды рендерятся прямо на картинке). |
| `.cfg` | Открывает **Xilla Configuration Hub** — интерактивное inline-меню настроек (Модули, Библиотеки, Пакеты). |
| `.ai <вопрос>` | Задать вопрос встроенному ИИ (работает на базе `Gemini`). |
| `.igen <описание>` | Сгенерировать квадратную картинку по вашему промпту. |
| `.pack <ext>` | Упаковывает Python-модуль (`.py`) в любой текстовый формат (например `.txt`) для удобной передачи. |
| `.unpack` | Распаковывает Xilla Pack обратно в рабочий модуль. |
| `.zip <реплай>` | Запаковать файл или директорию в архив (`.zip`, `.tar`, `.tgz`, `.bz2`). |
| `.update` | Безопасное обновление юзербота прямо из Telegram с красивой анимацией загрузки. |

*Внимание: Xilla использует механизм "Мандаторных обновлений". Если на GitHub выходит критический патч, бот попросит вас прописать `.update` перед продолжением работы для вашей безопасности.*
